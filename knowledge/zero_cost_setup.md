# Guia: Seu Blog Online com Custo Zero (GitHub Pages)

Como você já está usando o **GitHub Desktop**, publicar seus artigos é mais fácil do que postar no Facebook! Siga este guia simples:

---

## 1. Organizando suas Pastas
Dentro da pasta do seu projeto (`escritor-blog-master`), crie uma nova pasta chamada **`blog`**. É lá que você vai colocar todos os artigos que baixar da ferramenta.

## 2. O Passo a Passo (Tiazinha Style)

1.  **Gere o Texto**: No Escritor Master, digite o tema e clique em "Gerar".
2.  **Mídia (Fotos/Vídeos)**: No Arquiteto, coloque o link do YouTube e o nome da sua foto.
3.  **Baixe a Página**: Clique em **"Baixar Página"**. O arquivo vai para a sua pasta de "Downloads".
4.  **Mova o Arquivo**: Arraste esse arquivo para a pasta `blog` que você criou.

### 📸 Como lidar com as Fotos:
Para que suas fotos apareçam no site, você precisa:
1.  Criar uma pasta chamada `imagens` dentro da pasta principal do seu projeto.
2.  Colocar suas fotos lá dentro.
3.  No Arquiteto, no campo de imagem, use o caminho: `imagens/nome-da-foto.jpg`.

### 🎬 Como lidar com os Vídeos:
Não suba vídeos no GitHub!
1.  Suba no **YouTube**.
2.  Copie o link que aparece lá em cima no navegador.
3.  Cole no campo "Link do Vídeo" no nosso Arquiteto. Ele cuidará de fazer o vídeo aparecer certinho.

## 3. GitHub Desktop:
1.  Abra o programa GitHub Desktop.
2.  Ele vai mostrar que tem um arquivo HTML novo e uma foto nova na pasta `imagens`.
3.  Escreva "Novo Post" no campo de baixo.
4.  Clique em **"Commit to main"**.
5.  Depois clique no botão lá em cima: **"Push origin"**.

---

> [!TIP]
> **Dica de Ouro**: Você pode renomear o arquivo baixado para algo simples antes de subir, ex: `receita-de-bolo.html`. Isso ajuda o Google a te encontrar!

> [!IMPORTANT]
> **Sem WordPress, sem boletos**: Lembre-se, esse método é 100% gratuito para sempre. O GitHub é o "patrão" que hospeda seu site de graça porque ele é uma ferramenta para programadores, e agora você é uma também! 🚀✍️🛡️
